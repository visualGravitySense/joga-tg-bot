# 🧘 Yoga Learning Bot

Telegram бот для изучения йоги с использованием открытых данных и персонализированными рекомендациями.

## 🌟 Особенности

- 📚 **Каталог поз йоги** - более 10 базовых поз с подробными описаниями
- 🎯 **Персональные рекомендации** - на основе уровня пользователя
- 📊 **Отслеживание прогресса** - статистика занятий и достижений
- 🧘 **Поза дня** - ежедневные рекомендации для практики
- ⚙️ **Настройки профиля** - выбор уровня сложности
- ⭐ **Система оценок** - для улучшения рекомендаций

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка бота

1. Создайте бота в Telegram через [@BotFather](https://t.me/BotFather)
2. Получите токен бота
3. Создайте файл `.env` на основе `.env.example`:

```env
BOT_TOKEN=your_bot_token_here
ADMIN_USER_ID=your_telegram_user_id
```

### 3. Запуск бота

```bash
python bot.py
```

## 🎯 Функциональность

### Основные команды

- `/start` - начать работу с ботом
- `/help` - справка по командам
- `/settings` - настройки профиля
- `/progress` - ваш прогресс
- `/pose` - случайная поза
- `/daily` - поза дня

### Интерактивные кнопки

- 🧘 **Поза дня** - ежедневная рекомендация
- 📚 **Каталог поз** - поиск по категориям
- 📊 **Мой прогресс** - статистика занятий
- ⚙️ **Настройки** - уровень сложности
- 🎯 **Рекомендации** - персональные советы

### Категории поз

- **Стоячие** - Тадасана, Врикшасана, Вирабхадрасана I
- **Сидячие** - Падмасана
- **На коленях** - Марджариасана, Баласана
- **Лежачие** - Бхуджангасана, Сету Бандхасана
- **Перевернутые** - Адхо Мукха Шванасана
- **Расслабляющие** - Шавасана

## 🗄️ База данных

Бот использует SQLite для хранения:

- **Пользователи** - профили и настройки
- **Позы йоги** - каталог с описаниями
- **Сессии** - история занятий
- **Прогресс** - статистика и достижения

## 🔧 Технологии

- **Python 3.8+**
- **aiogram 3.2.0** - современная библиотека для Telegram Bot API
- **SQLite** - легковесная база данных
- **asyncio** - асинхронное программирование
