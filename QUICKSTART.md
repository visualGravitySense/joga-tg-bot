# 🚀 Быстрый старт Yoga Learning Bot

## 📋 Что нужно для запуска

1. **Python 3.8+** установлен на вашем компьютере
2. **Telegram аккаунт** для создания бота
3. **5 минут** свободного времени

## ⚡ Быстрая установка (3 шага)

### Шаг 1: Создание бота в Telegram

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Введите имя бота (например: "My Yoga Bot")
4. Введите username бота (например: "my_yoga_bot")
5. **Скопируйте токен** (выглядит как `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Шаг 2: Настройка проекта

```bash
# Клонируйте или скачайте проект
cd yoga-bot

# Создайте файл .env с вашим токеном
echo "BOT_TOKEN=ваш_токен_здесь" > .env
echo "ADMIN_USER_ID=ваш_telegram_id" >> .env
```

### Шаг 3: Запуск

```bash
# Установите зависимости и запустите бота
python deploy.py
```

## 🎯 Готово! 

Теперь найдите вашего бота в Telegram и отправьте `/start`

## 📱 Основные команды бота

- `/start` - начать работу
- `/help` - справка
- `/daily` - поза дня
- `/pose` - случайная поза
- `/progress` - ваш прогресс
- `/settings` - настройки

## 🔧 Если что-то не работает

### Ошибка "BOT_TOKEN не установлен"
```bash
# Проверьте файл .env
cat .env
# Должно содержать: BOT_TOKEN=ваш_токен
```

### Ошибка "Module not found"
```bash
# Установите зависимости
pip install -r requirements.txt
```

### Ошибка подключения к Telegram
- Проверьте правильность токена
- Убедитесь, что интернет работает
- Попробуйте перезапустить бота
