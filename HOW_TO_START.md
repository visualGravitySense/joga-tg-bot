# 🧘 Как начать заниматься йогой с ботом

## 🚀 Быстрый старт

### 1. Найдите бота в Telegram
- **Username**: @aawraBot
- **Ссылка**: [https://t.me/aawraBot](https://t.me/aawraBot)

### 2. Отправьте команду `/start`
Бот приветствует вас и показывает главное меню с кнопками.

### 3. Выберите свой уровень
- **🟢 Начинающий** - если вы новичок в йоге
- **🟡 Средний** - если у вас есть опыт
- **🔴 Продвинутый** - если вы опытный практик

## 📱 Команды бота

- `/start` - начать работу с ботом
- `/help` - справка по командам
- `/daily` - поза дня
- `/pose` - случайная поза
- `/progress` - ваш прогресс
- `/settings` - настройки профиля

## 🧘‍♀️ Рекомендуемый план занятий

### Для начинающих (1-2 недели)
1. **День 1-3**: Поза горы (Тадасана) - 5 минут
2. **День 4-6**: Поза ребенка (Баласана) - 5 минут
3. **День 7-10**: Поза кошки (Марджариасана) - 5 минут
4. **День 11-14**: Комбинация поз - 10 минут

### Для среднего уровня (2-4 недели)
1. **Неделя 1**: Стоячие позы + расслабление
2. **Неделя 2**: Позы на коленях + прогибы
3. **Неделя 3**: Балансовые позы
4. **Неделя 4**: Комплексная практика

### Для продвинутых
- Ежедневная практика 20-30 минут
- Комбинация различных типов поз
- Фокус на сложных асанах

## 💡 Советы для начинающих

### ⏰ Время практики
- **Лучшее время**: утром или вечером
- **Продолжительность**: начните с 5-10 минут
- **Регулярность**: лучше 5 минут каждый день, чем час раз в неделю

### 🏠 Место для практики
- Тихое, спокойное место
- Достаточно места для движений
- Коврик для йоги (опционально)

### 👕 Одежда
- Удобная, не стесняющая движений
- Дышащие материалы
- Босиком или в носках

### 🧘‍♀️ Подход к практике
- **Не спешите** - йога не соревнование
- **Слушайте тело** - не делайте через боль
- **Дышите** - дыхание важнее позы
- **Будьте терпеливы** - прогресс приходит со временем

## 🆘 Если что-то не работает

### Бот не отвечает
1. Проверьте интернет-соединение
2. Перезапустите Telegram
3. Попробуйте команду `/start` снова

### Ошибки в боте
1. Отправьте `/help` для справки
2. Попробуйте команду `/settings`
3. Перезапустите бота командой `/start`

### Вопросы по йоге
- Используйте команду `/help`
- Изучите описания поз в каталоге
- Начните с простых поз
